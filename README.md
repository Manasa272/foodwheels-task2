# Internship Task 2 – FoodWheels

## Objective
Recreate Internship Task-1 using a strict tech stack.

## Tech Stack Used
- Templating Engine: Nunjucks
- Bundler / Build Tool: Vite

## Folder Structure
src/templates/ → Nunjucks templates  
dist/ → Compiled HTML output  

## How to Run

1. Install dependencies
```
npm install
```

2. Build project
```
npm run build
```

3. Open output
```
dist/index.html
```

## Output
The final HTML output matches the Internship Task-1 document exactly.
